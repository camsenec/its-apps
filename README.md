# ITS Applications (its-apps)
Implementations of
- CA (Cooperative Awareness)
- DEN (Decentralized Environment Notification)
- RLT (Road and Lane Topology)
- TLM (Traffic Light Maneuver)
on top of ROS.

## Prerequisites
- CMake 3.5 or higher
- Vanetza 0.8 (https://www.vanetza.org)
